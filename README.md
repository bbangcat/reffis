# **Reffis Team**

## **Intro**

**`Reffis` 팀은 러스트 (또는 기타 언어)를 주로 사용하여 개발하는 Organization 입니다.**

또한, 파이썬 또는 자바스크립트로 디스코드 봇도 만들겁니다!

> `Reffis`: Rust Ferris를 조금 바꿔서 만든 이름입니다.
